# Live-Crib-Wall-Model
Live Crib Wall Model is a novel numerical model written in the statistical computing language R.
Live crib walls are soil and water bioengineering techniques built to restore and stabilise slopes subjected to landslides and erosion. This model helps design live crib walls, assesses their Internal Compound Stability as well as the stability of the slope in which they are built, and the hydro-mechanical effect of the vegetation growing within the live crib wall system.  
The model comprises six interrelated modules:
• Module i: slip geometry and crib wall design;
• Module ii: eco-hydrological variables estimator;
• Module iii: plant growth & timber decay predictor;
• Module iv: soil slice model;
• Module v: plant-soil slice model;
• Module vi: slope reinforcement & stability analysis. 
Each module should be ran sequentially. Model functions are in the file “live_crib_wall_model_functions.R”. 

The model is set up for a particular case study in NE Scotland. As such, sample inputs are kindly provided and specific assumptions are set. 

Users must redefine their path files for loading inputs, saving outputs, and calling for the required functions.
